About Me

I am a recent graduate from a 6 month, full-time full stack engineering course, looking for a Junior Software Engineering position.

I enjoy communicating and working with others, and I love solving puzzles.

Before this career change, I was in digital marketing, communications, and sales.

Projects

Full Stack Online Store

I built both the frontend and backend for a fictional online store called Daintree. I used React for the front end, and Ruby on Rails for the back end. The site included:

  • filter functionality
  • add to and remove from cart
  • login/logout and authorized access
  • API requests

Web Scraper

I built a CLI application using Ruby. This application scraped the website seek.com for Australian Software Engineer positions.

Netflix Clone

I used React to build an app that made an API request to The Movie Database, and displayed the information in a Netflix-like format. Movies could be favourited, and added/removed from a favourited list.

Photo Finder

I built this webapp using JavaScript, HTML, and CSS. Using the search bar sent an API request to Unsplash, and then displayed the photos using CSS grid.
